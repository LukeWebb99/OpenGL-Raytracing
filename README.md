# OpenGL-Raytracing

A real time ray-tracing demo using OpenGL and compute shaders

Features include:

- PBR
- Global illumination
- Lights
  - Directional 
  - Point
  - Spot
- Shadows
- Environment Map
- Camera system
- Basic GUI
- Post Processing
